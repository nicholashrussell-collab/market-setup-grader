# Market Setup Grader v7.4

Simplified paper-live dashboard. The main page now focuses on the live scanner, paper portfolio, ranked stock list, chart inspection, activity log, and cloud scan saves. The older research/backtester lab is still available at `/research`.

This app does not place real broker orders. It is paper/signals only.

## Deploy flow

Copy this version into the existing GitHub project folder, commit, push, and Vercel will redeploy automatically.
